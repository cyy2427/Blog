# helloflask

本项目实现了一个简易的web博客应用，实现功能包括：

- [ ] 用户管理 
  - [x] 用户注册
  - [x] 用户登录与登出
  - [ ] 用户个人中心
    - [x] 头像上传和显示
    - [ ] 个人资料设置

- [ ] 内容发布
  - [x] 文章发布（含标题、集成CKEditor富文本编辑）
  - [ ] 文章展示
  - [ ] 文章修改和删除
  - [x] 短文本发布
  - [x] 短文本展示（详情+列表）
  - [ ] 文章修改和删除

- [ ] 内容互动与反馈
  - [x] 评论发布与展示（列表）
  - [ ] 评论删除
  - [ ] 点赞
